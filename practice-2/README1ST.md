# Practice 2

This is a practice exercise to build a tag cloud.

The starting files (index.html, style.css) are in the "problem" folder, and they come with HTML markup and existing CSS styles for the UI styles.

You shouldn't need to make any changes in the HTML, and you only need to worry about the layout styles for the tags.

Mockups for how the final layout should look are in the "mockups" folder.

## Functional Requirements

The tags should be displayed in a "cloud" format as shown in the mockup.

The tags should have 10px of space between them, wrap onto multiple lines, and be horizontally centered in the container.

If you get stuck, the completed files are in the "solution" folder.
